# NPC-MagicalGirlProject

동덕여자대학교 중앙 게임 개발 동아리 ‘NPC’의 여름방학 단기 프로젝트에서 완성된 모바일 게임입니다.

Unity(C#)으로 개발되었습니다.

기획 1명, 디자인 2명, 개발 3명의 팀원이 참여했습니다.
